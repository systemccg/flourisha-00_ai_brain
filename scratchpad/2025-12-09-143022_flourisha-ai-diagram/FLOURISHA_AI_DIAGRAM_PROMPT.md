# Flourisha AI System Architecture Diagram Prompt

**Purpose:** Image generation prompt for visualizing Flourisha AI's complete operational system flow.

**Use:** Supply this prompt to image generation tools (DALL-E, Midjourney, Leonardo, etc.) to create the architectural visualization.

---

## COMPREHENSIVE DIAGRAM PROMPT

```
Create a 3D Pixar-style illustration of Flourisha AI's circular system architecture diagram.

CHARACTER & COMPOSITION:
- Feature Flourisha (3D Pixar style, confident and supportive pose) at the center
- Flourisha's hands cradle and support a glowing circular system diagram above/around her
- Label the center "Flourisha AI"
- Surround Flourisha with three key elements orbiting around her:
  * CORE Context
  * Skill System
  * Planning Engine

OUTER RING - Five Interconnected Stages Flowing Clockwise:

1. CONTENT INGESTION 
   Input Sources displayed:
   - Documents
   - PDFs
   - YouTube Videos
   - Web Content
   - User Input

   Mini Process Flow showing the journey:
   Ingest → Extract → Validate → Synthesize → Design

   Arrow flowing inward labeled "Raw Information"

2. KNOWLEDGE HUB
   Three Parallel Storage Systems visible and distinct:
   * Vector Store (semantic search, embeddings) - visualize as networked nodes
   * Graph Store (relationships, Neo4j/Graphiti) - visualize as interconnected web
   * Whole Document Store (complete context preservation) - visualize as organized library

   AI Brain documentation layer shown below storage systems
   Arrow flowing to next stage labeled "Structured Knowledge"

3. STRATEGIC COMMAND
   Visual representation:
   - Brain icon (intelligence processing)
   - Crown icon (leadership/decision authority)
   - Decision node with branching paths visible

   Arrow flowing down labeled "Orchestrated Tasks"

4. AGENTIC OPERATIONS 
   Multiple specialized agent types branching outward:
   - Engineer Agent (code implementation) - shown with code symbols
   - Architect Agent (system design) - shown with blueprint symbols
   - Researcher Agent (investigation) - shown with magnifying glass
   - Designer Agent (UX/product) - shown with palette/design symbols
   - Pentester Agent (security) - shown with shield symbols

   Parallel execution arrows radiating outward
   Output arrow labeled "Completed Work"

5. SYSTEM EVOLUTION
   Feedback loops flowing back to center/Flourisha AI:
   - Updated Documentation icon
   - New Skill Patterns icon
   - Learned Behaviors icon
   - Enhanced Context icon

   Dual Source of Truth visible:
   - Google Drive sync (real-time, shown as cloud)
   - GitHub sync (historical, shown as code repo)

   Synchronization symbols between the two sources
   Arrow flowing back to center labeled "System Learning"

CONNECTING ELEMENTS & FLOW:
- Thick glowing arrows showing primary clockwise flow direction
- Dotted luminescent feedback lines showing learning loops returning to center
- All arrows have subtle motion/energy glow

COLOR CODING SCHEME:
- Blue: Data flows (Content Ingestion → Knowledge Hub)
- Green: Processing layers (Validation, Synthesis)
- Gold: Intelligence nodes (CORE, Planning, Strategic Command)
- Purple: Action/Execution (Agentic Operations)
- Orange: Evolution/Learning (System Evolution)

DATA SYNCHRONIZATION:
- Symbols showing Google Drive ↔ GitHub bidirectional sync
- Real-time notation on Google Drive connection
- Historical/Version Control notation on GitHub connection
- Subtle glow indicating active synchronization

VISUAL STYLING:
- Modern, clean tech aesthetic with Pixar warmth
- Circular/cyclic design emphasizing continuous operation and feedback
- 3D depth with layered transparency to show system overlap
- Semi-transparent layers showing how different systems interact
- Subtle motion/energy visualization (particle effects or light trails) showing data flowing through the system
- Glowing nodes and pathways to emphasize active connections
- Professional yet friendly color palette appropriate to Pixar style
- Flourisha's hands naturally supporting/cradling the diagram from below
```

## RELATED CONTEXT

This diagram represents the complete Flourisha AI operational architecture:
- **Input**: Raw content from multiple sources
- **Processing**: Knowledge structuring and intelligent storage
- **Decision**: Strategic orchestration layer
- **Execution**: Parallel agentic operations
- **Evolution**: Continuous system learning and improvement

The circular design emphasizes that this is a **continuous feedback loop** where system evolution feeds back into improved future operations.
